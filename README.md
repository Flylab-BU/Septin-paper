# Septin-paper
This repository compiles the scripts used in the septin project.

Read preprint: [Septin Complexes Regulate Microtubule Organization and Synaptic Function at the Neuromuscular Junction.](https://www.biorxiv.org/content/10.64898/2026.03.01.708893v1.abstract)

Requirements:

Python 3.9 using lifelines (v0.27.4) and  pandas (v1.4.2) matplotlib (v3.5.2) and R (v4.4.1).
